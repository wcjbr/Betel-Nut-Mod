# Changelog

## [3.0.0] - 成瘾 HUD 可视化系统

### 新增

- 新增槟榔成瘾 HUD，可在游戏内直观看到当前依赖状态。
- 新增成瘾阶段显示，包括轻度依赖、中度依赖、重度依赖、严重依赖和极度依赖。
- 新增戒断倒计时显示，玩家可以看到距离下一次戒断反应的剩余时间。
- 新增 HUD 危险提示效果，在高依赖阶段或即将触发戒断时提供更明显的视觉反馈。
- 新增 HUD 位置自定义功能，玩家可以调整成瘾 HUD 的显示位置。
- 新增 HUD 拖动编辑模式，玩家可以直接用鼠标拖动 HUD 并保存位置。
- 新增客户端 HUD 配置，支持保存玩家自己的 HUD 显示位置。
- 新增相关语言文件文本，用于 HUD、提示信息和按键说明。

### 调整

- 优化成瘾阶段与戒断反应之间的对应关系。
- 优化 HUD 显示逻辑，使玩家更容易判断当前成瘾程度。
- 优化戒断倒计时同步逻辑，避免 HUD 显示和服务端实际戒断逻辑不一致。
- 优化 HUD 默认位置，减少与原版生命值、饥饿值和经验条的重叠。
- 优化槟榔类物品 tooltip 风险提示。

### 修复

- 修复 HUD 显示轻度依赖时，实际触发最高级戒断反应的问题。
- 修复 HUD 倒计时归零后，戒断反应没有正常触发的问题。
- 修复 HUD 位置不理想、容易遮挡原版 UI 的问题。
- 修复部分客户端 HUD 数据在重进世界或状态变化后显示不准确的问题。

### 说明

- HUD 设置属于客户端本地配置，不会影响服务器成瘾、戒断和惩罚逻辑。
- 本次更新主要改善玩家对成瘾机制的理解和反馈体验。
- 没有 Farmer's Delight Refabricated 时，HUD 系统仍然可以正常工作。

## [1.0.0] - 初始发布

### 新增

- 新增槟榔树系统：
  - 加入槟榔树原木、木板、楼梯、台阶、栅栏、栅栏门、按钮、压力板、树叶和树苗。
  - 支持槟榔树苗种植与自然生成。
  - 槟榔树可生成生槟榔、槟榔叶和树苗等资源。

- 新增槟榔物品：
  - 生槟榔。
  - 熟槟榔。
  - 槟榔叶。
  - 多种口味槟榔与进阶槟榔物品。

- 新增营火烘烤：
  - 生槟榔可通过营火烘烤为熟槟榔。
  - 熟槟榔作为后续口味加工的基础材料。

- 新增口味槟榔：
  - 辛辣槟榔。
  - 甜味槟榔。
  - 清凉槟榔。
  - 夜行槟榔。
  - 提神槟榔。
  - 蜂蜜槟榔。
  - 荧光槟榔。
  - 幻翼槟榔。
  - 末影槟榔。
  - 青金石槟榔。
  - 石英槟榔。
  - 岩浆槟榔。
  - 紫水晶槟榔。
  - 合成天下槟榔、琳琅天下槟榔、地下槟榔等扩展槟榔。

- 新增成瘾系统：
  - 食用槟榔会增加玩家成瘾值。
  - 不同槟榔可提供不同增益效果，并拥有不同成瘾增量。
  - 成瘾数据通过 Cardinal Components API 保存。

- 新增戒断惩罚：
  - 长时间未食用槟榔后，玩家会根据成瘾状态逐步积累戒断值。
  - 戒断阶段会施加移动速度降低、挖掘速度降低、饥饿、虚弱、反胃、黑暗等惩罚效果。

- 新增死亡后保留成瘾：
  - 玩家死亡后可保留成瘾与戒断数据。
  - 支持复活后延迟重新应用戒断状态。
  - 是否保留成瘾、是否保留戒断、是否复活后重应用均可配置。

- 新增进食限制：
  - 高戒断阶段会限制玩家使用部分食物或恢复类物品。
  - 牛奶、金苹果等恢复物品可按配置保留特殊处理。

- 新增最大生命值惩罚：
  - 戒断达到较高阶段后，会通过属性修饰器降低玩家最大生命值。
  - 惩罚会随戒断阶段提升，并在戒断缓解后移除。

- 新增 Farmer's Delight Refabricated 厨具联动：
  - 安装 Farmer's Delight Refabricated 时，加载 Cooking Pot 与 Cutting Board 联动配方。
  - 对应口味槟榔可通过 Farmer's Delight 厨具路线制作。
  - 未安装 Farmer's Delight Refabricated 时，原版工作台配方仍正常可用。
  - 安装 Farmer's Delight Refabricated 时，已有厨具替代路线的原版工作台口味槟榔配方会被条件隐藏。

- 新增农民村民交易：
  - 农民村民可交易生槟榔、槟榔叶、熟槟榔、部分口味槟榔和合成天下槟榔。
  - 交易开关、价格和数量可配置。

- 新增配置系统：
  - 生成 `betel_nut.json` 配置文件。
  - 支持配置成瘾、戒断、生命值惩罚、进食限制、槟榔树自然生成、末影槟榔传送和村民交易等功能。
  - 配置会进行基础校验与默认值迁移。

- 新增调试命令：
  - `/betel addiction get`：查看当前成瘾、戒断、计时与生命值惩罚状态。
  - `/betel addiction set <value>`：设置成瘾值。
  - `/betel addiction set <value> reset_timer`：设置成瘾值并重置上次食用计时。
  - `/betel addiction add <value>`：增加成瘾值。
  - `/betel addiction clear`：清空成瘾与戒断状态。
  - `/betel withdrawal set <value>`：设置戒断值。
  - `/betel withdrawal trigger`：快速触发戒断测试。
  - `/betel eatingtest`：检查当前手持物品是否会受进食限制影响。
  - `/betel tree generate`：在当前位置附近生成一棵槟榔树用于测试。
  - `/betel ender teleport`：测试末影槟榔随机传送逻辑。
  - `/betel trades info`：查看农民村民交易配置摘要。
  - `/betel reload`：重新加载槟榔模组配置。

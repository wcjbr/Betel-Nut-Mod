 # Betel Nut Mod 3.0.0 更新说明

本次更新重点加入了槟榔成瘾 HUD 可视化系统。玩家现在可以在游戏中直接查看当前依赖阶段、成瘾进度和戒断倒计时，不再需要猜测自己什么时候会触发戒断反应。

同时，本版本加入了 HUD 位置自定义与拖动编辑模式。玩家可以根据自己的界面布局调整 HUD 位置，设置会保存在本地客户端，不会影响服务器逻辑。

主要内容：

- 新增成瘾 HUD 显示。 
- 新增成瘾阶段与戒断倒计时。
- 新增 HUD 危险提示。
- 新增 HUD 位置自定义。
- 新增 HUD 拖动编辑模式。
- 修复 HUD 阶段显示与戒断惩罚不一致的问题。
- 修复倒计时归零但戒断反应未触发的问题。
- 优化 tooltip 风险提示。

注意：

本模组中的槟榔相关内容仅作为游戏机制与玩法创作使用，不代表现实健康建议，也不鼓励现实中食用槟榔。
